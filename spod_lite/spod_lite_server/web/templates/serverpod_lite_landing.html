Build a client in three lines.

The spod_lite_sdk package wraps the typed Serverpod client with auth sugar. One package. Every Flutter platform.

lib/main.dart 
import 'package:spod_lite_sdk/spod_lite_sdk.dart';

final spod = SpodLite('{{apiUrl}}');
await spod.auth.signInAsAdmin('you@example.com', 'password');

// Typed — your IDE knows every field
final posts = await spod.client.posts.listPosts();
